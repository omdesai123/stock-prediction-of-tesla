# 🚗 TSLA Stock Movement Predictor

A machine learning web app that predicts whether Tesla (TSLA) stock will go **UP** or **DOWN** the next day, built with Python, Flask, and a modern dark-themed frontend.

---

## 📁 Project Structure

```
project/
├── model.py              # Data fetching, feature engineering, model training
├── app.py                # Flask backend & prediction API
├── model.pkl             # Saved trained model (generated by model.py)
├── templates/
│   └── index.html        # Frontend HTML (Jinja2 template)
└── static/
    └── style.css         # Styling for the web UI
```

---

## ⚙️ Requirements

- Python 3.10+
- pip

Install all dependencies with:

```bash
pip install yfinance scikit-learn flask pandas numpy
```

If you have multiple Python versions, use the full path:

```bash
C:\Users\<you>\AppData\Local\Programs\Python\Python310\python.exe -m pip install yfinance scikit-learn flask pandas numpy
```

## 🚀 Getting Started

### Step 1 — Train the Model

Run `model.py` to fetch TSLA historical data, engineer features, train the RandomForest classifier, and save `model.pkl`:

## 🧠 How It Works

### Features Used for Prediction

| Feature | Description |
|---|---|
| `MA10` | 10-day rolling average of closing price (USD) |
| `MA50` | 50-day rolling average of closing price (USD) |
| `Daily Return` | Today's percentage change (e.g. `0.023` = +2.3%) |
| `Avg Vol 7` | 7-day rolling average of trading volume |
| `MA Ratio` | Momentum indicator — `MA10 ÷ MA50` |

### Target Variable

- `1` = Stock closes **higher** tomorrow (UP)
- `0` = Stock closes **lower** tomorrow (DOWN)

### Model

- **Algorithm:** RandomForestClassifier (200 trees, max depth 6)
- **Split:** 80% train / 20% test (chronological, no shuffle)
- **Library:** scikit-learn

## 🖥️ Using the Web App

1. Open `http://127.0.0.1:5000`
2. Enter today's values for all 5 features
3. Click **Run Prediction →**
4. See the UP/DOWN forecast and confidence probability bar

> **Tip:** You can find current MA10, MA50, and volume data on [finance.yahoo.com](https://finance.yahoo.com/quote/TSLA/) or any stock charting tool.

## 🛠️ Troubleshooting

| Problem | Fix |
|---|---|
| `ModuleNotFoundError` | Run `python -m pip install <module>` using your full Python path |
| `TemplateNotFound: index.html` | Make sure `index.html` is inside a `templates/` folder |
| `style.css` not loading | Make sure `style.css` is inside a `static/` folder |
| Port already in use | Change port in `app.py`: `app.run(port=5001)` |
